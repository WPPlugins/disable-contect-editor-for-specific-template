== Description ==

Disable content editor plugin remove the page content editor for specific page template

= PHP Requirements =
Disable content editor plugin 5.2 and newer require PHP5.3
Disable content editor plugin  5.1.1 and older require PHP5.2

== Installation ==



This section describes how to install the plugin and get it working.

1. Activate the plugin through the 'Plugins' add new  menu in WordPress

2. After activation the plugin option show on above setting option
3. Go to disable plugin editor and select template name from dropdown and click submit button
